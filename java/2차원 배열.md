## 1) 2차원 배열 선언과 할당 방법

- 배열 선언

```java
자료형[][] 배열명;
자료형 배열명[][];
자료형[] 배열명[];
```

- 배열 할당

```java
자료형[][] 배열명 = new 자료형[행크기][열크기];
자료형 배열명[][] = new 자료형[행크기][열크기];
자료형[] 배열명[] = new 자료형[행크기][열크기];
```

## 3) 2차원 배열의 초기화 방법들

```java
1) for문을 이용한 초기화
    for(int i=0; i<arr.length; i++) {
        for(int j=0; j<arr[i].length; j++){
            arr[i][j] = j;
        }
    }
```
